Open index.html

## To rebuild the app:

1. Delete 'distr' folder

2. Install devDependencies:

```
npm install
```

2. Build the app:
```
gulp build
```
Open index.html


## Serving changes:

The browser reloads, sass files compile, js transpiles to es5, and images minify on change.

```
gulp serve
```
or just
```
gulp
```
since 'serve' is set as a default gulp task.
